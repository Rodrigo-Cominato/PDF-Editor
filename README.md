# RodrigoPDF

This repository contains the source code for RodrigoPDF, a browser-based PDF editing tool inspired by iLovePDF.

Features:

- Merge PDFs
- Split PDFs
- Rotate Pages
- Reorder Pages (Drag & Drop)
- Remove Pages
- Extract Pages
- Add Page Numbers
- Add Watermarks

Additional features include:

- Light & Dark Mode
- Portuguese & English
