RodrigoPDF

This repository contains the source code for RodrigoPDF, a browser-based PDF editing tool inspired by iLovePDF.

RodrigoPDF allows users to edit PDF files directly in their browser, with all processing performed locally for maximum privacy.

Features
Merge PDFs
Split PDFs
Rotate Pages
Reorder Pages (Drag & Drop)
Remove Pages
Extract Pages
Add Page Numbers
Add Watermarks

Additional features include:

Light & Dark Mode
Portuguese & English
